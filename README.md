# linux-labs

My hands-on labs while studying Linux fundamentals. Each folder has:
- `task.md` — the lab itself, tasks only, no answers
- `notes.md` — my commands, output, and answers after doing the lab

## Labs

| Lab | Topic | Status |
|---|---|---|
| [01](./lab-01-users-permissions/) | Users, permissions, filesystem basics |  Complete |

## How I work through a lab

1. Read `task.md`, try each section from memory first.
2. Run everything on a disposable VM (never on a real/production machine).
3. Copy real commands and output into `notes.md`.
